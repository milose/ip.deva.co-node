# ip.deva.co

> Simple IP resolving service.

## Quick Start

```sh
curl https://ip.deva.co
# or
curl http://ip.deva.co
```

## Api

### JSON Output

```sh
curl https://ip.deva.co
# or
curl http://ip.deva.co
```

### Plain Text Output

```sh
curl https://ip.deva.co?text
# or
curl http://ip.deva.co?text
```
